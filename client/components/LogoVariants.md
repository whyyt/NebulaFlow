# NebulaFlow Logo 设计方案

## 方案1: 向上扩散的能量弧线

### 设计理念
- **视觉表现**: 4条从中心点向上扩散的弧线，形成扇形扩散效果
- **力量感**: 表现"成长、扩散、上升"的能量
- **光束变化**: 
  - 最外层弧线最粗（2.5px），透明度最高（0.9）
  - 向内逐层变细、透明度递减
  - 所有弧线从底部中心点（100, 160）出发，向上扩散
  - 形成从密集到扩散的视觉节奏

### 适用场景
- 强调"成长"和"扩散"概念
- 适合表现平台的发展和影响力扩大

### SVG 特点
- 4条对称的弧线
- 使用贝塞尔曲线（Q命令）创建平滑弧线
- 从底部中心向上扇形扩散

## 方案2: 螺旋流动轨迹

### 设计理念
- **视觉表现**: 3条螺旋上升的光束，形成动态流动感
- **力量感**: 表现"动能、前进、持续运动"的动感
- **光束变化**:
  - 最外层螺旋最粗（2.5px），路径最长
  - 中间层中等粗细（2px）
  - 最内层最细（1.8px），形成紧凑的螺旋
  - 所有光束从不同起点开始，最终汇聚到中心上方
  - 形成螺旋上升的视觉节奏

### 适用场景
- 强调"动能"和"持续前进"
- 适合表现平台的活力和动态发展

### SVG 特点
- 3条不同长度的螺旋路径
- 使用多个贝塞尔曲线点创建流畅的螺旋
- 从底部两侧向中心上方螺旋汇聚

---

## 方案3: 汇聚的能量流

### 设计理念
- **视觉表现**: 4条光束从不同方向汇聚，然后向上流动
- **力量感**: 表现"汇聚、整合、上升"的能量
- **光束变化**:
  - 左右对称的两对光束
  - 外侧光束较粗（2.5px），路径更长
  - 内侧光束较细（2px），路径更短
  - 所有光束从底部两侧出发，向中心汇聚后向上流动
  - 形成汇聚后上升的视觉节奏

### 适用场景
- 强调"汇聚"和"整合"概念
- 适合表现平台的聚合力和上升趋势

### SVG 特点
- 4条对称的光束路径
- 使用贝塞尔曲线创建平滑的汇聚和上升轨迹
- 从底部两侧向中心上方汇聚流动

## 使用建议

### 在首页使用
```tsx
import { Logo } from "../components/Logo";

// 在标题上方使用
<Logo variant={1} size={200} style={{ color: "#ffffff" }} />
<h1>On-Chain Growth</h1>
```

### 在导航栏使用
```tsx
<Logo variant={1} size={32} style={{ color: "#ffffff" }} />
<span>NebulaFlow</span>
```

### 颜色适配
- 深色背景：使用 `color: "#ffffff"` 或 `color: "currentColor"`
- 浅色背景：使用 `color: "#000000"` 或深色渐变

### 尺寸建议
- 主视觉：200-300px
- 导航栏：32-48px
- 图标：24-32px
- 卡片：80-120px

## 技术细节

### SVG 优化
- 使用 `currentColor` 继承父元素颜色
- 使用 `strokeLinecap="round"` 创建圆润的端点
- 通过 `opacity` 控制透明度层次
- 通过 `strokeWidth` 控制线条粗细层次

### 响应式
- 使用 `size` prop 控制整体尺寸
- SVG viewBox 固定为 "0 0 200 200"，确保比例不变
- 可通过 CSS 进一步控制样式
